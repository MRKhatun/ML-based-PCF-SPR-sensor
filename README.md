Analyte identification in conventional photonic crystal fiber surface plasmon resonance (PCF-SPR) sensors typically relies on explicit resonance wavelength and spectral measurements. Such approaches can be challenging when analytes are closely spaced or when spectral data is incomplete. To address this, we present a classification-oriented machine learning (ML) framework integrated with a dual-side D-shaped PCF-SPR sensor for direct analyte identification from spectral responses. The sensor employs optimized air-hole geometry and a gold plasmonic coating to enhance mode confinement and analyte–metal interaction. Spectral responses and the corresponding dataset were generated using COMSOL Multiphysics finite-element simulations. These data were subsequently used to train and evaluate the ML classifier without relying on explicit resonance peak extraction. The proposed framework achieves a classification accuracy of approximately 95% and an ROC-AUC of 0.996 using the XGBoost model. Shapley additive explanation (SHAP) analysis provides interpretability by identifying the spectral features that drive classification decisions. The framework is demonstrated on closely spaced ester analytes (ethyl acetate and butyl acetate). It can be applied to experimental measurements and adapted to other chemical or biosensing applications, offering a generalizable and interpretable methodology for next-generation PCF-SPR–based analyte detection.
